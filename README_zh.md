# 智驾时代：Agent进化简史

> 从Prompt到自进化的AI驾驶之路
> 作者：**Yason** · MindApex · VokoForge

## 简介

一辆从手摇启动到自动驾驶的进化之路。本书用自动驾驶的隐喻，将Agent从Prompt工程到Loop自进化的完整历程娓娓道来。每一章都是一个驾驶场景，让你在理解Agent的同时，也看到AI的"驾驶"技术如何一步步走向成熟。
