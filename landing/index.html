Evolution

五代演化

从手摇启动到自进化车队——不是替代，而是层层叠加。就像现代汽车依然有方向盘和刹车。

01
Prompt Engineering · 手摇启动
2022–2023
你得亲自摇手柄。写一段提示词，AI 才动一下。你说一句它做一步，不催它就停在那。最早的汽车，每次启动都得下来摇半天。
02
Context Engineering · 装上导航
2023–2024
给 AI 装了一张地图。它能看到项目文档、历史记录、相关资料，而不是凭空瞎猜。但方向盘还在你手里，它只是看得更清楚了。
03
Harness Engineering · 装上刹车和方向盘
2024–2025
AI 开始真正做事了。能改代码、能跑命令、能发邮件。但能做事的同时也可能闯祸。于是 Harness 出现了：给它装刹车、装方向盘、装仪表盘。从"能做事"变成"能安全地做事"。
04
Loop Engineering · 自己会跑了
2025–2026
最关键的跃迁。AI 做完一件事不再停下来等你——它自己发现任务、执行、验证、记录、进入下一轮。就像定速巡航加自动导航：你设定目的地，它自己开。
05
Auto-Organization · 组队进化
2026+
一辆车跑得快，一支车队能运更多货。多个 Agent 分工协作：写代码的、做测试的、审查的。它们自动组队、自动协作，甚至演化出新的组织形态。
The Core

核心公式

Agent = LLM + Harness
!

没有 Harness 的 LLM

一个装在罐子里的大脑。聪明，但什么也干不了。也正是因为什么也干不了，所以也闯不了祸。

?

Harness 是什么

方向盘、刹车、交规、导航。一整套让 AI 安全可靠干活的工程系统。大模型决定了能力上限，Harness 决定了安全下限。

V

LLM + Harness = Agent

大脑加上身体。能上路了。能自己读文件、写代码、跑测试、发邮件——而且在规则之内，不会乱来。

Deep Dive

三重悖论

Agent 越强，越要回答这三个"反过来"的问题。而且它们拆不开。

1

记忆悖论

MEMORY PARADOX

记得越多不等于越聪明。信息塞太满，有用的那条反而不显眼。记忆要管理——存多久、怎么隔离、何时丢弃。业界现在拼的不是谁记得多，而是"谁忘得聪明"。

2

推理悖论

REASONING PARADOX

我们用 Harness 这套脚手架补偿基础模型推理能力的不足。但脚手架越复杂，它自己制造的新故障就越多。补一个洞，可能在别处戳出三个洞。失败不是线性累加的——它是雪球式复合的。

3

进化悖论

EVOLUTION PARADOX

让 Agent 自己改自己，不等于它会变好。自己评自己，分数只会越判越高——这叫退化循环。没有外部独立验证的进化，本质上是在一个越来越小的圈子里打转。
