💡 学习建议

这个学习路径的时间是**参考值**，不是硬性要求。如果你有编程基础，可以跳过某些阶段；如果你是纯小白，也可以放慢节奏。关键是**动手实践**——光看不练假把式，每学一个概念都要亲手写代码试试。

## B.1 零基础入门（第 1-2 周）

第 1-2 周 🚗 认识 AI，学会用 AI

**目标：**搞懂大模型是什么，能熟练使用 AI 工具，完成第一个小项目

- 了解大模型基本原理：什么是 LLM？它为什么能"说话"？
- 学习 Prompt Engineering 基础：角色设定、任务描述、输出格式
- 上手一个 AI 编程助手：Cursor 或 Claude Code，熟悉基本操作
- 理解 Token、上下文窗口、Temperature 等基础概念
- 尝试不同的 Prompt 技巧：Few-shot、CoT（思维链）、角色扮演
- 完成第一个小任务：用 AI 写一个简单网页（个人介绍页）

**产出物：**一个用 AI 辅助完成的静态网页 + 你的 Prompt 笔记

🔑 关键心态

这个阶段最重要的不是学多少技术，而是**建立对 AI 的"体感"**。多跟 AI 聊天，多试试不同的问法，感受它的能力边界。不要怕问"傻问题"——你跟 AI 聊得越多，对它的理解就越深。

## B.2 进阶学习（第 3-4 周）

第 3-4 周 🧭 给 AI 装上地图和眼睛

**目标：**理解 RAG 和 Agent 的核心概念，能写一个简单的 Agent

- 学习 RAG 基本概念：什么是向量？什么是 Embedding？为什么需要向量数据库？
- 动手搭一个最简单的 RAG：用 Chroma + OpenAI API 做文档问答
- 了解 Agent 基本架构：思考 → 行动 → 观察 的循环
- 学习 ReAct 模式：推理与行动交替进行
- 尝试写一个简单的 Agent：用 LangChain 实现一个能搜索的助手
- 学会用 AGENTS.md 管理项目：写一份给 AI 看的项目说明
- 理解 Function Calling：大模型怎么调用外部工具

**产出物：**一个能回答文档问题的 RAG 应用 + 一个简单的搜索 Agent

⚠️ 注意事项

这个阶段最容易陷入**"框架焦虑"**——今天学 LangChain，明天听说 CrewAI 火了又去学 CrewAI，后天 AutoGen 又火了……记住：框架只是工具，**核心概念才是根本**。把一个框架学透，比浅尝辄止学十个框架有用得多。

## B.3 实战提升（第 5-8 周）

第 5-8 周 🛠️ 从玩具到产品

**目标：**能独立构建一个完整可用的 Agent 项目，理解工程化要点

- 深入学习 Harness Engineering：权限控制、验证机制、失败恢复
- 搭建第一个完整的 Agent 项目：从需求分析到上线运行
- 学习可观测性：用 LangSmith 或 Langfuse 监控 Agent 行为
- 学习评估方法：怎么判断 Agent 做得好不好？
- 尝试多 Agent 协作：用 CrewAI 或 LangGraph 实现双 Agent 协作
- 理解沙箱机制：为什么 Agent 需要安全执行环境？
- 学习 Prompt 迭代优化：怎么系统性地提升 Agent 表现？
- 写一份完整的项目文档：技术方案、踩坑记录、优化思路

**产出物：**一个完整可用的 Agent 项目 + 项目文档 + 可观测面板

✅ 里程碑

完成这个阶段，你就已经**超过了 80% 自称"会 Agent"的人**。很多人停留在"能跑个 Demo"的阶段，但真正把 Agent 做成可用的产品，需要考虑的东西远比想象中多。能走到这一步，你已经具备了独立做 Agent 项目的能力。

## B.4 高手之路（第 9-12 周）

第 9-12 周 🚀 从单体到生态

**目标：**掌握高级 Agent 架构，形成自己的方法论

- 研究 Loop Engineering：Agent 循环的设计模式与最佳实践
- 设计 Agent Mesh 架构：多 Agent 网络的组织与协作方式
- 探索自进化系统设计：Agent 怎么从经验中学习和改进？
- 深入理解 MCP 协议：工具生态的标准化之路
- 研究 Worktree 机制：多工作区隔离与并行开发
- 学习成本优化：怎么在保证效果的前提下降低 Token 消耗？
- 关注前沿研究：World Model、Agentic AI 等新方向
- 形成自己的方法论：总结你自己的 Agent 设计原则

**产出物：**一个多 Agent 协作系统 + 你的 Agent 方法论笔记

🔬 进阶方向

到了这个阶段，没有标准答案了。你可以根据自己的兴趣选择方向：喜欢架构的可以深入研究多 Agent 系统，喜欢产品的可以专注于 Agent UX，喜欢算法的可以研究 Agent 评估和优化。**找到自己的细分领域深耕**，比什么都学一点更有价值。

## B.5 推荐资源清单

学习路上，好的资源能帮你少走很多弯路。以下是经过筛选的优质资源，按类别整理。

#### 📚 推荐书籍

- 《智驾时代：Agent 进化简史》
- 你正在读的这本书！用智能驾驶比喻讲清楚 Agent 的来龙去脉
- 《深度学习》（花书）
- 想深入理解大模型原理的同学可以读，有一定数学门槛
- 《Building Agentic Systems》
- LangChain 团队出品，系统讲解 Agent 系统构建，英文原版
- 《Prompt Engineering for Generative AI》
- Prompt Engineering 的经典入门书，基础概念讲得很清楚

#### 📝 推荐博客 / 公众号

- AI观察室（公众号）
- 本书作者的官方账号，持续更新 Agent 技术干货和行业洞察
- LangChain Blog
- LangChain 官方博客，有大量 Agent 最佳实践和案例分析
- Andrej Karpathy Blog
- AI 大神的博客，文章深入浅出，值得反复读
- 机器之心 / 量子位（公众号）
- 国内 AI 领域的头部媒体，跟进行业动态必备
- Hugging Face Blog
- 开源模型生态的风向标，了解最新模型和技术趋势

#### 💻 推荐开源项目

- LangChain / LangGraph
- 最流行的 Agent 开发框架，生态丰富，入门首选
- CrewAI
- 多 Agent 协作框架，角色分工清晰，上手简单
- AutoGPT
- 自主 Agent 的鼻祖项目，虽然不一定适合生产，但学习价值很高
- Dify
- 国产低代码 Agent 平台，可视化操作，非技术人员也能用
- MCP (Model Context Protocol)
- Anthropic 推出的工具接入协议，正在成为行业标准
- LlamaIndex
- RAG 领域的专业框架，数据连接能力超强

#### 🎓 推荐课程

- 吴恩达《Agentic AI》系列课程
- DeepLearning.AI 出品，从 Prompt 到 Agent 系统学习，英文配中字
- OpenAI Developer Tutorials
- OpenAI 官方开发者教程，官方出品，质量有保证
- Harvard CS50 AI
- 哈佛大学的 AI 入门课，零基础友好，系统性强
- fast.ai 课程
- 实战导向的深度学习课程，从项目出发反向学习理论

💡 资源使用建议

资源不在多而在精。与其收藏一百个公众号，不如**把一个好博客的文章读完**。与其看十个教程，不如**跟着一个教程动手做完**。学习最忌讳的就是"收藏即学会"——打开收藏夹吃灰，不如现在就动手写一行代码。
